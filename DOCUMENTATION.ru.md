### 3. API.md - Полный справочник API

**Назначение**: Детальная документация функции за функцией по API

**Содержимое**:
- **Пакет streebog**:
  - `Sum256()` - хеширование на 256 бит
  - `Sum512()` - хеширование на 512 бит

- **Пакет gost3410**:
  - Перечисление `Curve` (все 8 кривых ТК26)
  - Перечисление `HashID` (HashAuto, Streebog256, Streebog512)
  - Тип `PrivKey` и методы:
    - `NewPrivKey()` - Генерация нового ключа
    - `FromRawPriv()` - Создание из байт
    - `Public()` - Вывод открытого ключа
    - `Sign()` - Подписание дайджеста
  - Тип `PubKey` и методы:
    - `Verify()` - Проверка подписи
    - `ToCompressed()` - Сериализация сжата
    - `ToUncompressed()` - Сериализация несжата
    - `FromCompressed()` - Восстановление из сжата
    - `FromUncompressed()` - Восстановление из несжата

- **Пакет gostcrypto**:
  - Тип `Options` для конфигурации подписания
  - `Sign()` - Подписание сообщения (высокоуровневое)
  - `Verify()` - Проверка подписи сообщения (высокоуровневое)

- **Пакет kdf/hd**:
  - `Master()` - Генерирование главного ключа из семени
  - `Derive()` - Вывод дочерних ключей по путям

- **Обработка ошибок**:
  - Таблица распространённых типов ошибок
  - Значения и рекомендуемые действия

- **Константы**:
  - Размеры алгоритмов хеширования
  - Размеры подписей для каждой кривой
  - Размеры открытых ключей (сжато/несжато)

- **Гарантии безопасности потоков**
- **Советы по производительности**
- **Примечания совместимости**

**Размер файла**: ~11 КБ
**Целевая аудитория**: Пользователи API и разработчики

| Сжато | 33 байта | 32 байта | 65 байт | 64 байта |
| Несжато | 65 байт | 64 байта | 129 байт | 128 байт |

|--------|----------------|

| TC26_512_A/B/C | 128 байт (r \|\| s, по 64 байта каждый) |
